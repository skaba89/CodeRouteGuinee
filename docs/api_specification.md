# Specification API initiale

## Endpoints MVP

- `GET /health` : verifier l'etat de l'API.
- `GET /api/v1/dashboard` : indicateurs administratifs initiaux.
- `GET /api/v1/centers` : liste des centres agrees.

## Endpoints cibles

- `POST /api/v1/auth/login`
- `POST /api/v1/candidates`
- `GET /api/v1/candidates/{id}`
- `POST /api/v1/centers`
- `POST /api/v1/questions`
- `POST /api/v1/sessions`
- `POST /api/v1/exams/start`
- `POST /api/v1/exams/submit`
- `GET /api/v1/results/{reference}`
