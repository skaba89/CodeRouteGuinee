# Rapprochement financier CodeRoute Guinee

Ce module de supervision permet aux administrateurs nationaux de consulter les paiements, de filtrer les transactions et de suivre les exports financiers.

## Points couverts

- Consultation des paiements par operateur, statut et periode.
- Limitation du nombre de lignes retournees pour proteger l'interface.
- Journalisation des consultations via les logs d'audit.
- Base pour le suivi des transactions en echec et des anomalies de paiement.

## Prochaine evolution

Ajouter une vue frontend dediee pour afficher le detail des transactions, les paiements echoues et les alertes suspectes.
